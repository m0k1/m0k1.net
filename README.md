# m0k1.pw
This is my portoflio/blog site built using [Hexo](https://hexo.io/).  


## How to deploy ?
Check if you have [NodeJS](https://nodejs.org/en/) and  [Hexo](https://hexo.io/docs/index.html#Install-Hexo) installed
```bash
apt install libpng-dev  
npm install hexo -g
npm install
hexo generate  
hexo deploy  
```
